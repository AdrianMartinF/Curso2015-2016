Dataset Requirements

turnosmadrid_guardia.csv

  • Data are in the smart cities domain.

    • Los datos pertenecen al dominio de las ciudades inteligentes, en concreto, a la ciudad de Madrid.

  • Data are available as a CSV file.

    • La datsheet se encuentra en formato .csv

  • Data have an open license so it can be published.

    • La licencia puede encontrarse en este link.
    La datasheet se puede obtener en la página web de datos abiertos de Madrid.
    Es posible utilizar los datos mientras se cite la fuente de los mismos, y se mencione la fecha de última modificación.
    Está expresamente prohibido realizar labores de re-identificación de personas a partir de los datos.

  • Data can be easily linked with generic real-world entities (e.g., locations).

    • Los datos pueden ser linkeados con entidades mediante direcciones postales y coordenadas.
